# Электронная библиотека (АИС)

Электронная библиотека — это автоматизированная информационная система (АИС) каталогизации книг, разработанная в качестве экзаменационного проекта. Приложение позволяет управлять каталогом литературы, осуществлять гибкий поиск, фильтрацию, а также оставлять рецензии к книгам с поддержкой форматирования текста.

---

**Функциональные требования:**
* Реализация системы авторизации и разграничения прав доступа (RBAC: Администратор, Модератор, Пользователь).
* Ведение каталога книг с поддержкой фильтрации по жанрам.
* Система рецензирования с использованием языка разметки Markdown.
* Создание и наполнение личных коллекций (подборок) книг пользователями.

## 🛠 Стек технологий
* **Python 3.10**
* **Flask** — веб-фреймворк
* **Flask-SQLAlchemy** — ORM для работы с БД
* **Flask-Login** — управление сессиями
* **Bootstrap 5** — дизайн интерфейса
* **SQLite** — база данных
* **EasyMDE** — Markdown-редактор

## 🔐 Данные для проверки (Аккаунты)
Для демонстрации работы системы используйте следующие учетные данные:

| Роль | Логин | Пароль | Описание прав |
| :--- | :--- | :--- | :--- |
| **Администратор** | `admin` | `password123` | Полный доступ: управление пользователями, книгами, жанрами. |
| **Модератор** | `moderator` | `password123` | Управление контентом: добавление и редактирование книг. |
| **Пользователь** | `user` | `password123` | Просмотр каталога, написание рецензий, создание подборок. |
